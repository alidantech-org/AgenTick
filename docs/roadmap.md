# Roadmap

## Milestone 1 — local foundation

- `init`, `verify`, `watch`, `status`, `history`, and local skill listing;
- safe Git-root discovery and idempotent `.gitignore` updates;
- YAML schemas, JSONL event history, SQLite index, and local dashboard;
- protected-file findings and command evidence;
- deterministic skill bundles plus initial `pull` and `push` clients.

## Milestone 2 — audit engine

- protected-file content snapshots and automatic restoration evidence;
- machine-readable task scope and plan drift;
- Git diff and commit evidence;
- structured finding responses and human-resolved disputes;
- project graph generation and plugin rule API.

## Milestone 3 — skill registry

- account signup and website sessions;
- `login`, `logout`, and `whoami` with operating-system credential storage;
- public/private skill browsing and immutable version publishing;
- namespace membership, token scopes, revocation, and moderation;
- registry-side bundle validation and version-range resolution.

## Milestone 4 — release and hosted platform

- production remote libSQL deployment;
- optional object storage for large immutable bundles;
- signed releases, staged npm publishing, provenance, and audit reports;
- package release management across the pnpm workspace.
